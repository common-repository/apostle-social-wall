Apostle Social Wall is a great plugin for adding the apostle connect social wall to your website.

== Description ==

Apostle connect is an app for retrieving, sharing and posting authentic social media messages via the profiles of colleagues!From this app Apostle will provide the customer a RSS-feed to show the posts on a social wall on the website.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/apostle-social-wall` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Done!

== Frequently Asked Questions ==
 
= How to display feed at frontside using this plugin? =

First Go to Admin -> Social Wall Feed -> Add New Feed, than add title and apostle connect social wall feed url.after save feed, you can see Shortcode , copy that and paste in any page.
